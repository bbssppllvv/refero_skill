# Anti-AI-Slop Manifesto

**CRITICAL: Your design must NOT look AI-generated.**

AI-generated interfaces converge on the same tired patterns because they optimize for "safe" and "average." Real designers make intentional, contextual choices.

## Experiment vs. Safe

**"Best practices" can be a trap.**

AI models default to the most common patterns because they're trained on averages. But the most memorable designs often break conventions intentionally.

**Safe (forgettable):**
- 3-column pricing grid because "everyone does it"
- Hero with left text + right image because "it works"
- Card grid with equal spacing because "it's clean"

**Intentional (memorable):**
- 2-column pricing because your product only has 2 tiers
- Full-width hero image with overlay because it fits the brand
- Asymmetrical layout because you want visual tension

**The difference:** Can you justify the choice? If yesâ€”do it. If noâ€”question it.

> "I chose this because [specific reason for THIS project]" beats "I chose this because everyone does it."
